# 心理帳戶計算機

依「心理帳戶」分法分配每月收入，並可逐月記帳、追蹤每個帳戶花了多少、還剩多少。

- 純靜態、**無任何外部連線**（字型、圖示、圓環圖全部內建）→ 可完全離線使用
- 響應式設計，手機 / 平板 / 桌機與獨立 App 視窗皆已調校
- 可安裝為 Mac App（獨立視窗、Dock 圖示）
- 記帳資料**儲存於你這台裝置的瀏覽器**（localStorage），不會上傳到任何地方

---

## 兩大分頁

### 1. 分配計算
輸入月收入後，**六個帳戶滑桿都可調整**，總計恆維持 100%：

- 調整「投資」以外的任一帳戶 → 差額全由**投資帳戶**吸收，其餘帳戶不動（投資被吸到 0、已分配滿 100% 時，滑桿會卡在上限）。
- 調整**投資帳戶** → 剩餘比例平均分配給 教育 / 玩樂 / 給予 / 長期儲蓄；**生活帳戶維持不變**。

生活、投資帳戶另有理想目標提示（達標 / 超標）。

### 2. 記帳紀錄
在分配計算頁設定好月份與比例後，按「**儲存本月分配**」即建立一張當月記帳單。切到「記帳紀錄」分頁即可：

- 上方選擇月份（可存多個月份）、刪除整月
- 摘要：本月收入 / 已花費 / 剩餘
- 每個帳戶顯示「預算金額」「已花費」「剩餘可用」與進度條，超支會變紅
- 在每個帳戶下方逐筆輸入花費（品項 + 金額），即時更新剩餘；可刪除單筆

> 同一個月份再次「儲存本月分配」會更新預算比例，但**已記錄的花費會保留**。

## 檔案結構

```
income-allocator/
├── index.html              ← 主程式（CSS / JS 全內嵌，單檔即可運作）
├── manifest.webmanifest    ← App 安裝設定
├── sw.js                   ← 離線快取（Service Worker）
└── icons/                  ← App 圖示
```

之後只要維護 `index.html` 一個檔案即可。

---

## 一、用瀏覽器開啟
直接**雙擊 `index.html`**，或拖進瀏覽器即可。沒有外部依賴，斷網也能用。

## 二、在 Mac 上像 App 一樣開啟

**方法 A：Safari → 加入 Dock（最簡單，免架站）**
1. 用 **Safari** 開啟 `index.html`（macOS 14 Sonoma 以上）
2. 上方選單列 **檔案 → 加入 Dock…**
3. Dock 會出現錢包圖示，點開即為獨立 App 視窗

**方法 B：Chrome / Edge 安裝（含離線快取，需用網址開啟）**
```bash
cd income-allocator
python3 -m http.server 8000
```
用 Chrome 開啟 <http://localhost:8000> → 點網址列右側「安裝」圖示。App 內也有「安裝為 App」按鈕會顯示對應步驟。

## 三、放到網路上，隨時隨地記帳（推薦）
把整個 `income-allocator/` 資料夾上傳到免費靜態空間，就能用一個網址在手機 / 電腦隨開隨記：
- **GitHub Pages**、**Netlify**（可直接把資料夾拖進 app.netlify.com/drop）、**Cloudflare Pages**、**Vercel** 皆可，全部免費。
- 上傳後用該網址開啟即可；在手機上還能「加入主畫面」變成 App。

> 提醒：記帳資料存在「你開啟用的那支瀏覽器」裡。若要長期記帳，固定用同一支瀏覽器 / 同一個網址開啟，資料才會一直都在。

## 四、（原本的本機方式，選用）
整個資料夾上傳 GitHub Pages / Netlify / Cloudflare Pages / Vercel 等靜態空間即可。

---

## 關於資料儲存
記帳資料存在開啟此工具的「該瀏覽器 / 該裝置」中。換瀏覽器、換電腦、或清除瀏覽器資料時不會同步，也不會自動備份。若要在多裝置共用，需自行另接雲端（目前版本不含此功能）。
